│
├── 生命周期 (Lifetimes)
│   ├── 生命周期注解
│   └── 借用检查器

Week 2:

## 阶段二：异步 Rust 和 Tokio（1 周）

### 学习目标
- 理解 Future 和 async/await
- 掌握 Tokio 运行时
- 能够编写异步程序

### 学习内容

#### Tokio 异步编程
```rust
// 异步 main 函数
#[tokio::main]
async fn main() {
    let (result1, result2) = tokio::join!(
        async_task_1(),
        async_task_2()
    );
}

// 异步函数
async fn fetch_data() -> Result<String, Error> {
    let response = reqwest::get("https://api.example.com/data").await?;
    let body = response.text().await?;
    Ok(body)
}
```

## 阶段三：Axum Web 框架（1 周）

### 学习目标
- 掌握 Axum 路由和处理器
- 理解中间件机制
- 能够构建 RESTful API

### 学习内容

#### Axum 基础
```rust
use axum::{
    routing::get,
    Router,
};

async fn handler() -> &'static str {
    "Hello, World!"
}

#[tokio::main]
async fn main() {
    let app = Router::new()
        .route("/", get(handler));

    let listener = tokio::net::TcpListener::bind("0.0.0.0:3000")
        .await
        .unwrap();
    
    axum::serve(listener, app).await.unwrap();
}
```

### 练习任务
- [ ] 实现一个 REST API（用户管理）
- [ ] 添加认证中间件
- [ ] 实现分页查询
